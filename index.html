Vito's Tracks

Here's just a simple application to visualiz my garmin data on a 3D canvas. Datas are autommaticly scraped from the garmin connect app using python and sélénium. All data are stored in a PostgreSQL database also using python. A Flask API provides all tracks in json format to this visualisation app using CesiumJS.
